旅游网站项目文档

1. 项目概述

这是一个现代化的旅游网站项目，包含以下主要功能：

  • 用户认证（登录/注册）
  • 景点展示和搜索
  • 旅游攻略
  • 用户评价系统
  • 盲盒旅游功能
  • 响应式设计

2. 技术栈

  • 前端：HTML5, CSS3, JavaScript (ES6+)
  • 后端：Node.js, Express
  • 数据库：MongoDB
  • 部署：PM2, Nginx

4. 页面功能说明

4.1 首页 (index.html)

  • 响应式导航栏
  • 轮播图展示
  • 热门景点推荐
  • 搜索功能
  • 登录/注册入口

4.2 认证系统

  • 用户登录
  • 用户注册
  • 手机验证码
  • 密码强度检测
  • 记住密码功能
  • 第三方登录

4.3 景点展示

  • 景点列表
  • 筛选功能
  • 排序功能
  • 分页功能
  • 景点详情

4.4 盲盒旅游

  • 盲盒类型选择
  • 开盒动画
  • 结果展示
  • 预订功能

6. 部署说明

环境要求：

  • Node.js 16.x+
  • MongoDB 4.4+
  • PM2
  • Nginx

部署步骤：

  1. 安装依赖：npm install
  2. 配置环境变量：修改 .env 文件
  3. 启动MongoDB服务
  4. 使用PM2启动Node服务
  5. 配置Nginx反向代理

7. 注意事项

  • 确保MongoDB服务正常运行
  • 检查端口占用情况
  • 注意文件权限设置
  • 定期备份数据库
